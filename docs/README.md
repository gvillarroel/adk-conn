# ADK Connector Docs

This documentation set explains how the `adk-conn` repository is organized and how the
connector agent should be operated during early development.

## Sections

- [Connector Overview](./overview.md)
- [Runtime and Configuration](./runtime-and-configuration.md)

## Current Scope

The repository currently contains a minimal Google ADK example that wires one connector
tool into a conversational agent. The docs focus on the practical decisions needed to
turn that example into a reusable connector package.
